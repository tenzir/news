type: feature
author: gitryder
created: 2025-12-17T13:24:10Z
---

After creating a Node, the Tenzir UI now has an additional "other" tab next to the existing
"docker" and "linux" options. This tab shows that Node's Tenzir Token directly, making
deployment in cloud environments more convenient.
