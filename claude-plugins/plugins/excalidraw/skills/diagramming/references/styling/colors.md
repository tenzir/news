# Colors

Color properties and defaults.

## Properties

```json
{
  "strokeColor": "#1e1e1e",
  "backgroundColor": "transparent",
  "opacity": 100
}
```

## Defaults

| Property          | Default       |
| ----------------- | ------------- |
| `strokeColor`     | `#1e1e1e`     |
| `backgroundColor` | `transparent` |
| `opacity`         | `100`         |

## Format

Use 6-character hex (e.g., `#1971c2`) or `transparent`.

## Opacity

Range: 0-100 (integer).
